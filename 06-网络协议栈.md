# 06-Linux 内核网络协议栈

> 版本：v7.0-rc4  
> 更新时间：2026-03-20

---

## 📋 概述

Linux 网络协议栈负责：
- 网络数据包处理
- 协议实现 (TCP/IP/UDP)
- 路由和转发
- 套接字接口
- 网络设备管理

struct sk_buff {
    union {
        struct {
            struct sk_buff *next;     // 下一包

## 🎯 实践练习

### 练习 1: 简单网络模块

```c
// net_module.c
#include <linux/module.h>
#include <linux/kernel.h>
#include <linux/netdevice.h>
#include <linux/etherdevice.h>

static struct net_device *my_netdev;

static netdev_tx_t my_start_xmit(struct sk_buff *skb,
                                  struct net_device *dev)
{
    // 发送数据包 (环回示例)
    dev_kfree_skb(skb);
    dev->stats.tx_packets++;
    dev->stats.tx_bytes += skb->len;
    return NETDEV_TX_OK;
}

static const struct net_device_ops my_netdev_ops = {
    .ndo_init = NULL,
    .ndo_open = NULL,
    .ndo_stop = NULL,
    .ndo_start_xmit = my_start_xmit,
};

static int __init net_init(void)
{
    // 分配网络设备
    my_netdev = alloc_etherdev(0);
    if (!my_netdev)
        return -ENOMEM;
    
    // 设置操作
    my_netdev->netdev_ops = &my_netdev_ops;
    
    // 设置 MAC 地址
    eth_hw_addr_random(my_netdev);
    
    // 注册设备
    if (register_netdev(my_netdev)) {
        free_netdev(my_netdev);
        return -ENODEV;
    }
    
    pr_info("Network device registered: %s\n", my_netdev->name);
    return 0;
}

static void __exit net_exit(void)
{
    unregister_netdev(my_netdev);
    free_netdev(my_netdev);
    pr_info("Network device unregistered\n");
}

module_init(net_init);
module_exit(net_exit);
MODULE_LICENSE("GPL");
```

### 练习 2: Netfilter 钩子

```c
// netfilter_hook.c
#include <linux/module.h>
#include <linux/netfilter.h>
#include <linux/netfilter_ipv4.h>
#include <linux/ip.h>

static struct nf_hook_ops nfho;

static unsigned int hook_func(void *priv,
                              struct sk_buff *skb,
                              const struct nf_hook_state *state)
{
    struct iphdr *iph;
    
    // 获取 IP 头
    iph = ip_hdr(skb);
    if (!iph)
        return NF_ACCEPT;
    
    // 打印源和目的 IP
    pr_info("Packet: %pI4 -> %pI4\n", &iph->saddr, &iph->daddr);
    
    // 接受所有数据包
    return NF_ACCEPT;
}

static int __init nf_init(void)
{
    nfho.hook = hook_func;
    nfho.hooknum = NF_INET_LOCAL_IN;  // 本地输入
    nfho.pf = PF_INET;
    nfho.priority = NF_IP_PRI_FIRST;
    
    nf_register_net_hook(&init_net, &nfho);
    pr_info("Netfilter hook registered\n");
    return 0;
}

static void __exit nf_exit(void)
{
    nf_unregister_net_hook(&init_net, &nfho);
    pr_info("Netfilter hook unregistered\n");
}

module_init(nf_init);
module_exit(nf_exit);
MODULE_LICENSE("GPL");
```

## ✅ 总结

Linux 网络协议栈核心要点：

- **套接字层** - 用户空间接口
- **SKB 缓冲** - 数据包管理
- **网络设备** - 硬件抽象层
- **IP 协议** - 路由和转发
- **TCP/UDP** - 传输层协议
- **Netfilter** - 防火墙和 NAT
- **命名空间** - 网络虚拟化

网络协议栈是内核最繁忙的子系统之一！
