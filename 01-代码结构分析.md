## 🛠️ 构建系统

### 依赖要求
- **GNU Make**: >= 4.0
- **编译器**: GCC 或 Clang
- **Rust**: 可选 (用于 Rust 内核模块)

### 常用构建命令

```bash
# 查看帮助
make help

# 编译内核
make -j$(nproc)

# 编译模块
make modules

# 安装内核
make modules_install install

# 清理构建
make clean
make mrproper

# 生成文档
make htmldocs

# 代码检查
make C=1       # 检查重新编译的文件
make C=2       # 检查所有源文件

# Rust 代码检查
make CLIPPY=1
```

## 🌟 特色功能

### Rust 内核支持
从 6.1 版本开始，Linux 内核正式支持用 Rust 编写内核模块：
- `rust/` 目录包含 Rust 绑定和宏
- 提供内存安全的驱动开发选项
- 与 C 代码无缝互操作

### BPF (Berkeley Packet Filter)
- `kernel/bpf/` - BPF 虚拟机
- `net/bpf/` - 网络 BPF
- 用于网络过滤、跟踪、安全

### io_uring
- 高性能异步 I/O
- 零拷贝 I/O 操作
- 现代存储设备的优化
